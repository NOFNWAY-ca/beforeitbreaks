# AGENTS.md — TerraComm Timesheet

Client project for Erik at TerraComm Inc.

## What This Is

A branded timesheet app delivered as a single HTML file. Gerry's first paid-adjacent client work. The file is self-contained -- no server, no backend, no dependencies.

## Files

- TerraComm TimeTracker V2.html -- current deliverable
- TerraComm TimeTracker V1.html -- previous version, keep for reference

## Constraints

- Single HTML file, always. Client receives one file they can open in any browser.
- No frameworks, no build step, no external dependencies.
- Branded to TerraComm -- colors, logo usage, and naming should match their identity.
- Data stays local (localStorage or export-to-CSV) -- nothing leaves the browser.

## Delivery

Send the HTML file directly to Erik. No deployment needed.
