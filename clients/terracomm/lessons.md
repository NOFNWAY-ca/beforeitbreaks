## 2026-04-17 — Log table readability
When improving the TerraComm log, keep the workflow intact and layer clarity onto the existing cells instead of replacing controls. Project context now lives in the Project cell as subtext, while EEA and WBS stay present but visually subdued.

## 2026-04-17 — Version large TerraComm changes
When a TerraComm request expands from visual polish into workflow behavior, save it to a new versioned HTML file first. Use the new file as the active branch until Gerry decides it should replace the current deliverable.
