# CLAUDE.md — Septic Dispatch Demo

Generic septic operations demo project.

## What This Is

A dispatch management tool built as a demonstration project — intended to show prospective clients what a bespoke single-file HTML tool looks like in practice. Not currently in production use and not tied to a live client.

## Constraints

- Single HTML file. The whole point is showing a client they get one file that works offline.
- No frameworks, no build step, no external dependencies.
- Data is local only — nothing leaves the browser.

## Files

- `dispatch.html` — working dispatch manager / PT tracker demo
- `BRIEF.md` — planning reference for future septic-related adaptations

## Status

Generic demo. Safe to adapt or reuse for future septic-related client work.

## Design Notes

- Keep the branding neutral.
- Preserve the offline-first single-file pattern.
- Treat this as a demo template, not a canonical client deployment.
